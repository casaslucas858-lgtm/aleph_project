## 📋 ¿Qué es ALEPH?

ALEPH es una plataforma educativa que mide **comprensión profunda** en matemáticas, no memorización ni velocidad.

A diferencia de competencias tradicionales (Canguro, OMA, AMC), ALEPH prioriza:
- ✅ Demostración formal sobre cálculo rápido
- ✅ Rigor lógico sobre intuición visual
- ✅ Construcciones matemáticas sobre fórmulas memorizadas

## 🎯 Sistema de niveles

### **π (PI) - Progresión lineal**
Niveles automáticos sin competencia. Avanzás cuando resolvés 70% de los problemas.

| Nivel | Descripción |
|-------|-------------|
| **π=3** | Secundaria básica - Primeras demostraciones |
| **π=3.1** | Secundaria media - Inducción y lógica |
| **π=3.14** | Secundaria avanzada - Rigor creciente |
| **π=3.141** | Pre-universitario - Demostraciones clásicas |
| **π=3.1415** | Universitario temprano - Construcciones formales |

### **e (EULER) - Competitivo** *(Próximamente)*
Ranking mensual, degradación por inactividad.

### **ℵ (ALEPH) - Elite** *(Próximamente)*
ELO dual (Matemática + Algoritmos), duelos 1v1, torneos.

## 🚀 Cómo usar

1. **Registrate** en el link de arriba
2. **Resolvé problemas** de tu nivel actual
3. **Avanzá automáticamente** al resolver 70% correctamente
4. **Probá niveles superiores** cuando estés listo
